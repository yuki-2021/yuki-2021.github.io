    Css中布局机制有很多

    • 普通流 - 默认机制，左到右，上到下
    • 浮动
    • 定位(position)
    • 多列布局
    • flex布局
    • grid布局(网格)

    普通流

    默认机制，元素 从左到右，从上到下。

    浮动

    Float ，元素脱离普通流，不脱离文字流。

    • 浮动后，脱离文档流，不脱离文字流 - 文字环绕

    • 多元素浮动，紧跟同类，过长下移

    • 脱离文档流，造成高度塌陷，影响后面元素

    文字环绕 - 脱离文档流，不脱落文字流

    清除浮动

    有如下方法

    • 父元素overflow:hidden
    • 父元素内部添加div,设置clear
