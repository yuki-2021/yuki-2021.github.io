    切面

    由如下内容

    • @Pointcut - 定义切点
    • @Before - 前置通知
    • @Around - 环绕通知
    • @AfterReturning - 执行完毕
    • @After - 最后通知
    • @AfterThrowing - 异常通知

    切点表达式

    由如下符号

    * 匹配任意名称
    .* 匹配多级包
    .. 匹配任意参数

    exection - 匹配切点
