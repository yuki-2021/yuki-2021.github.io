分区和挂载

安装Linux后，会对磁盘分区，然后自动创建目录，最后把分区挂载在目录上。

挂载之后，用户可以通过操作系统提供的API来访问存储设备。

默认分区挂载方式

默认挂载方式如下

目录 磁盘
/boot 500M
/swap 2G
/root 剩余

比如，

  • 你的数据和程序放在/data和/web中。
  • 按照默认分区方式，一旦sda2坏了，你的/data和/web数据同时凉凉
  • 如果/data放在/sd3中，/web放在/sd4。/sd3坏掉了，那你的/sd4不会有任何影响。
  • 所以，需要根据具体情况，进行具体分区。

特点如下
| | NAT | Bridge | Internal | Host Only |
| —————— | —- | —— | ——– | ——— |
| 虚拟机 -> 主机 | √ | √ | × | 需要设置 |
| 主机 -> 虚拟机 | × | √ | × | 需要设置 |
| 虚拟机 -> 其他主机 | √ | √ | × | 需要设置 |
| 其他主机 -> 虚拟机 | × | √ | × | 需要设置 |
| 虚拟机之间 | × | √ | 同网络名称可以 | √ |
