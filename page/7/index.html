错误

mysqld: File ‘./binlog.index’ not found (OS errno 13 - Permission denied)

原因

之前使用强制用root用户运行Mysql，产生了一些属于root用户的日志。

当不强制用root用户后，mysql在加载这些文件时，没有权限

解决方法

定位错误

一个良好的习惯是，在启动时，就用tail -f观察/var/log/mysqld.log,随时了解启动情况。

问题如下

解决办法
  • 定位到文件
  • 发现是属主root，因为mysqld默认用mysql用户启动，导致打不开文件。
  • 使用chown解决或则chmod解决

DQL语句是Data Query Language，用于查询数据。

基础查询

基础的查询语法是

AS - 起别名
  • 给运算结果起别名
  • 给列起别名
  • 给表起别名
Distinct - 去重复
  • Distinct 紧跟在Select后面
  • 单列，一列作为整体，去重。多列，多列作为整体，去重。
分组 - group by
  • 一般配合聚合函数(count、max、min、avg、sum)，实现分组后统计
  • 多次使用，表示多个分组条件。比如，统计每个部门的每个工种的平均工资，就是部门和工种
Having - 子查询
  • having用来对group by结果进行过滤。
排序 - order by
  • 可选值有ASC、DESC。ASC可以省略。
  • 多次使用，先按c1排，然后按照c2排。
Limit - 分页
  • 语法是Limt(idx,count)。idx省略表示取前count条。
  • idx从0开始，比如取11-20条，应该是limit(10,10)。
