文件属性信息

在linux中，使用ls -li查看文件属性信息。

如下

  • Part I - Inode信息
  • Part II - 文件类型和权限信息
  • Part III - 硬链接个数
  • Part IV - 属主信息 和属组信息
  • Part V - 文件大小
  • Part VI - 修改日期
  • Part VII - 文件名称

Inode和Block

在linux中，文件由两部分组成。分别是Inode和Block。

  • Inode - 保存了文件名称、文件权限、文件内容的地址
  • Block - 保存了文件内容

两个文件的Inode相同，互为硬连接。

无法给目录创建硬连接。

Block对于磁盘IO效率影响？

  • 读取每个Block的时间是相同的。
  • Block小，IO效率会低，磁盘浪费小。
  • Block大，IO效率高，磁盘浪费多。

文件类型

文件类型，用来区分文件类型。

常见分类如下

  • - - 普通文件

  • d - 目录

  • l - 软连接

  • c/b - 设备文件/字符文件

  • s - socket文件

file - 查看文件类型

使用file查看文件类型。

设备文件、字符文件

b表示设备文件，c表示字符文件。

常用的字符文件如下

  • /dev/sda - 磁盘文件。
  • /dev/urandom - 字符文件，用于产生随机数。
  • /dev/zero - 字符文件，输入黑洞，可以用输入黑洞来填充文件。
  • /dev/null - 字符文件，输出黑洞，可以把输出信息重定向到输出黑的。
ln - 创建连接

ln表示创建链接。

  • -s - 表示创建软连接
df - 查看Block、Inode、分区信息

df 查看Inode或Block的使用情况。

  • -h - 人类可读方式
  • -i - 查看Inode信息

时间信息

文件时间分为三种

  • atime - 访问时间，Access Time。
  • Ctime - 修改时间，Change Time，表示修改Inode信息，比如名称、权限。
  • mtime - 修改时间，Modify Time，修改Block内容。
stat - 查看文件时间

stat用于查看文件时间相关信息。
