目录

切换目录 - cd

cd用于切换目录

查看路径 -pwd

pwd用于显示当前位置

目录大小 - du

du 用来统计文件大小，默认只显示目录大小。

  • -h - 人类可读方式
  • -a - 显示文件和目录大小
  • -s - 只显示总大小

目录或文件

文件创建 - touch

可以使用如下方式直接写入内容

另外，可以使用echo 来创建文件。

目录创建 - mkdir

mkdir用于创建目录

  • -p - 表示创建多级目录
查看 - ls
  • -a - 查看所有
  • -h - 人类可读
  • -S - 按照大小排序
  • -r - 顺序反转
  • -l - 详细信息
  • -A - 查看所有，不包括.和..
  • -t - 按照最近修改时间排序
  • -i - 显示inode
  • -d - 只查看目录

有地方可以用ll，是因为有alias(别名)。

  • ll 相当于ls -a
复制 - cp

cp用来复制文件或目录。

  • -r - 递归复制
  • -f - 强制执行
重命名 - mv

mv用来从命名和移动。

移动 - mv
删除 - rm

rm用来删除文件或目录。

  • f - 强制
  • r -递归

文件查看

查看 - cat
分页查看 - less

分页查看，按上下显示下一行。按pageDown/UP来翻页。

  • ↑↓ - 上下行
  • enter - 下一行
  • space - 下一页
  • pageDown / pageUp - 翻页
  • d / b - 半个屏幕行数
  • q - 退出
  • = - 显示你在什么位置
  • h - 显示帮助文档
  • / - 搜索
  • n/shift + n - 上一个、下一个
分页查看 - more

more和less类似，只是支持的功能比less少。

  • ↑↓ - 上下行
  • enter - 下一行
  • space - 下一页
  • pageDown / pageUp - 翻页
查看头部 - head

head用于查看前几行。

  • -n - 查看前n行
查看尾部 - tail

tail用于查看后n行。

  • -n - 后n行
  • -f - 持续监听

经常用来监听日志文件。比如监听mysql日志文件。

查找

命令查找 - which

用于查找命令。准确的说是在$PATH中查找可执行文件。

$PATH默认包含

文件查找 - whereis

查找文件和命令。准确的说是在$PATH中查找。

文件查找 - find

查找指定地方的文件。

  • -name - 文件名称，支持ant通配符
  • -type - 文件类型
  • -maxdepth/-mindepth - 最大和最小层级
  • -user/-group - 属组 /属主
  • -perm - 权限
  • -size - 按照大小查找

另外，还支持如下动作

  • -exec
  • -delete
