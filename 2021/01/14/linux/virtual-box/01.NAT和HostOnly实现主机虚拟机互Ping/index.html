Virtual Box四种网络模式

在Virtual Box中，有NAT(网络地址转换)、桥接模式(Bridge Adapter)、Internal(内部网络)、Host Only四种模式。

特点如下
| | NAT | Bridge | Internal | Host Only |
| —————— | —- | —— | ——– | ——— |
| 虚拟机 -> 主机 | √ | √ | × | 需要设置 |
| 主机 -> 虚拟机 | × | √ | × | 需要设置 |
| 虚拟机 -> 其他主机 | √ | √ | × | 需要设置 |
| 其他主机 -> 虚拟机 | × | √ | × | 需要设置 |
| 虚拟机之间 | × | √ | 同网络名称可以 | √ |
