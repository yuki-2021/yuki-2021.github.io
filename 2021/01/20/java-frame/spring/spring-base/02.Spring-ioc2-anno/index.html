作用域

@scope - 指定作用域

有两个属性

  • value - 表示scope范围，对于singleton和prototype不要配置。
  • proxyMode - 仅仅对于request和session范围有效

对于Value

范围 说明 常量
prototype 实例 ConfigurableBeanFactory.SCOPE_PROTOTYPE
singletom 单例 ConfigurableBeanFactory.SCIPE_SIMGLETON
request WebApplicationContext.SCOPE_REQUEST
session WebApplicationContext.SCOPE_SESSION

proxyMode主要可以返回一个代理类，主要有两个作用

  • 在Session或者request到来后，代理类动态创建target或者返回target。
  • 被singleton的对象引用，代理类是context初始化后就创建了，可以被singleton引用。
proxyMode jdk动态代理类 ScopedProxyMode.INTERFACES
proxyMode cglib代理类 ScopedProxyMode.TARGET_CLASS
