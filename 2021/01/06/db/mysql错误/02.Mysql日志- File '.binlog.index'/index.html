错误

mysqld: File ‘./binlog.index’ not found (OS errno 13 - Permission denied)

原因

之前使用强制用root用户运行Mysql，产生了一些属于root用户的日志。

当不强制用root用户后，mysql在加载这些文件时，没有权限

解决方法

定位错误

一个良好的习惯是，在启动时，就用tail -f观察/var/log/mysqld.log,随时了解启动情况。

问题如下

解决办法
  • 定位到文件
  • 发现是属主root，因为mysqld默认用mysql用户启动，导致打不开文件。
  • 使用chown解决或则chmod解决
