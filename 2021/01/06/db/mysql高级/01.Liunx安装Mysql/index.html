Mysql内容

  1. 配置文件在/etc/my.ini位置。

  2. 日志文件在/var/log/mysqld.log位置。

  3. 其他目录如下。可用rpm -ql 软件名称查看。

  4. mysqld用来启动服务器，mysql用来连接。
