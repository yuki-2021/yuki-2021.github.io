Java中的xml

在Java EE中，定义了xml模块，规定了xml解析的规范。

根据你的需求，你可以使用如下三种方式

  • 要解析整个xml，整个解析完成后，才能开始业务 - DOM解析器
  • 解析整个xml，边解析，边使用 - 给予流的stax/sax解析器
  • 只需要解析部分 - xpath解析器
