Spring概述

Spring是一个用于组件管理的容器。

  • 我们把配置信息提供给Spring，Spring把配置信息转换成BeanDefinition，帮我们完成依赖注入。这个过程叫做IOC。

  • Spring支持AOP。我们把横切逻辑放到Advice，Spring在创建Bean的时候，会采用gdk动态代理或者cglib方式，给我们返回代理类。

  • Spring支持组件扫描和自动注入和自动代理。我们不再需要自己提供配置信息，Spring自动完成组件扫描和依赖注入DI。

  • 使用Spring可以整合ORM、aspectj、spring mvc、Spring Transition和J2ee技术。

  • Spring程序支持多种单元测试框架，比如Junit，可以容易的获取到ApplicationContext。
