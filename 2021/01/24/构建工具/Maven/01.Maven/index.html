Maven

Maven是一个依赖管理和项目构建工具。功能如下

  • 可以进行依赖管理和自动更新
  • 约定了统一的项目结构
  • 提供了丰富的插件，可以用来构建、编译、测试、打包、生成报告等。

Maven生命周期和插件

生命周期、插件、目标(goal)

生命周期、插件、目标(goal)可以如下理解

  • 目标(goal) - 可以执行的一个功能点
  • 插件 - 多个goal(功能点)的集合
  • 生命周期 - 在执行构建命令时候，用来绑定插件
生命周期阶段

生命周期由如下阶段

阶段 作用 默认执行的插件
Clean 清除target目录 maven-clean-plugin
Validate 检查pom.xml是否正确
Compile build项目 maven-resource-plugin、maven-compiler-plugin
Test 单元测试 maven-surefire-plugin
Package 打包 maven-jar-plugin、maven-war-plugin
Verify 检查集成测试结果
Install 安装到本地仓库 maven-install-plugin
Site 生成项目信息 maven-project-info-reports-plugin
Deploy 发布到远程仓库 maven-deploy-plugin
构建项目三种方式
  • 执行生命周期 - 实际是执行默认插件
  • 执行具体插件(执行某个目标)
  • 插件的goal绑定到生命周期，然后执行生命周期
