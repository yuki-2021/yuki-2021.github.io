Vue组件

定义组件

在Vue中，组件分为全局组件和局部组件。

  • 全局组件 - 使用Vue.component()创建，挂载到Vue上，任何地方都能用。
  • 局部组件 - 使用components: {}引入，只有引入后，才能使用。

全局组件 - Vue.component

局部组件 - components

另外，你也可以手动指定组件名称。

组件名称

对于全局组件，名称建议中划线（-）分隔。

对于局部组件，建议

  • 声明时候，大拓峰格式
  • 引入时候，如果需要重新命名，可以自定义。不明名，会自动转换。

传值

传值 - props - 单向数据流

使用props在父子组件之间传值。

  • 父组件使用v-bind传值
  • 子组件使用props来接收

可以传递的类型有

  • 值
  • 对象
  • 传对象，接收值 – 自动解构

最后，也是最重要的，不要直接改变Props的值。

如果有需要，先让data=props，然后再修改。

非Props属性

这类属性

  • 在父组件传递值，子组件，不用props接收
  • 默认会添加到子组件最外层的元素上，使用inheritAttrs来控制是否添加。
  • 使用$attrs来访问

使用$attrs在vue和template中，都可以访问。

插槽 - Slot

<slot />和v-slot

插槽用于扩展组件。

  • 在子组件使用<v-slot name='xxx'/>定义插槽。不写name，默认是default。
  • 使用v-slot:xxx传入slot内容，或者使用#xxx。
  • 对于<slot />，不传递，默认展示<slot></slot>中的内容。

用<slot />定义插槽。

具名插槽

就是有名称的插槽。

  • 定义时，使用<slot name='xxx'/>指定名称
  • 传递内容，使用v-slot:xxx或者#xxx传递内容

动态组件

动态组件，用<component v-bind:is='xxx'>来标识。

在切换时候，需要保存组件的状态，使用<keep-alive></keep-alive />。它的原理是

  • 在切换时候，保存组件的状态
  • 切回来时，恢复状态
