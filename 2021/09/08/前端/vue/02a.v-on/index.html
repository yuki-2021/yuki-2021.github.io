    事件修饰符

    在js中，经常用到event.preventDefault()或event.stopPropagation()等事件方法。

    在Vue中，可以使用事件修饰符。

    • .prevent - 阻止默认行为
    • .stop - 阻止事件传播(用在内部)。对于冒泡浏览器(chrome)，阻止冒泡。对于捕获浏览器(ie)，阻止捕获。
    • .capture - 用在外部元素，这个元素(及内部)的事件模型，变成捕获模型。
    • .self - 自己生效(用在内部)
    • .once - 生效一次

    按键修饰符

    监听有两种按键修饰符。

    • 按键码
    • 按键名称

    按键码

    当使用键盘事件，需要按倒指定键，然后触发事件，否则不触发，可以使用按键码。

    常用按键码，有按键码别名。

    • .space、.tab、.enter - 空格、制表、回车
    • .delete (捕获“删除”和“退格”键)
    • .esc
    • .up 、.down、.left、.right
    • 其他的，直接用按键名称

    最后，使用Vue.config.keyCodes.x=x来定义不存在的按键别名。

    系统修饰键

    系统修饰符，对所有事件生效，同时按住按键，才能触发事件。

    • .ctrl
    • .alt
    • .shift
    • .meta - 在 Mac 系统键盘上，meta 对应 command 键 (⌘)。在 Windows 系统键盘 meta 对应 Windows 徽标键 (⊞)

    鼠标修饰符

    顾明思意，必须按下对的鼠标键，才会触发。

    • .left
    • .right
    • .middle
