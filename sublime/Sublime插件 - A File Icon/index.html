Sublime插件 - A File Icon

نُشر في 2020-12-25 عُدل في 2020-12-26

做项目的时候，发现了如下问题

  • 多了很多文件夹和文件

  • 需要来回切换

  • 很难直接看出需要的文件类型

于是安装了A File Icon，记录下安装过程。

  1. 打开命令面板(ctrl + shift + p)，输入 package Control，找到package Control:install package。
  2. 输入A File Icon，点击。
  3. 成功
