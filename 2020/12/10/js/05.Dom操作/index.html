Dom处理

Dom处理の笔记

DOM概述

在Java中，有需求是显示家族成员信息，这时候，对人进行抽象，提取出他们的特征，然后定义系列Java接口，然后实现。

同样，W3C进行了建模和接口定义过程，浏览器厂商负责实现过程。

  • 建的模型叫DOM树
  • 定义的接口叫DOM API

DOM API接口有

  • Node - 节点，用于遍历节点、Dom操作、属性
  • ChildNode - 表示子节点，只有</html>不是，用于Dom操作
  • Element、ParentNode - 用于遍历节点、Dom操作、获取Props、innerXXX、查询操作和位置属性
  • NodeList - 节点集合
  • HtmlCollection - Element集合
  • Text表示文本 - 获取属性、文本操作
  • NamedNodeMap - attr集合属性
  • Attr - 表示属性
  • CSSStyleDeclaration - css属性集合
  • DOMTokenList - className集合
  • DOMStringMap - data集合

Dom create

Dom提供了如下创建方法

  • document.createElement() - 创建节点
  • document.createTextNode() - 创建文本节点
  • document.createComment() - 创建注释
  • document.createDocumentFragment - 创建文档片段

插入大量元素时候,使用

  • Element.innerHTML
  • Element.innerText
  • Element.outerHTML
  • Element.outerText
  • Node.textContent

Dom遍历

首先，可以使用Node提供的遍历方法，得到NodeList

  • firstChild、lastChild、childNodes - 子节点
  • parentNode、ownerDocument - 父节点
  • previousSibling、nextSibling - 兄弟节点
  • hastChildNodes() - 是否有子节点

其次，可以使用Element和ParentElement提供的Api,得到的是HtmlCollection

  • firstElementChild、lastElementChild、children
  • parentNode、ownerDocument
  • previousElementSibling、nextElementSibling
  • childElementCount

调用Node.childNodes，得到NodeList，是动态集合、类数组对象

  • 获取每个Node,使用Node.childNodes[i] = Nodelist.prototype.item(i)
  • Nodelist.prototype.length - 长度
  • Nodelist.prototype.forEach() -遍历
  • Nodelist.prototype.keys() 、Nodelist.prototype.values() 、 Nodelist.prototype.entries()

调用Children,得到HtmlCollection，类数组、动态集合。

  • 获取元素 ParentNode.children[i] = ParentNode.children.item(i)
  • HTMLCollection.prototype.length
  • 根据id和name获取元素 HTMLCollection.prototype.namedItem()

其他创建HtmlCollection的方式有

  • getElementsByTagName() - 获取内容
  • document.all - 获取所有的id元素
  • document.scripts
  • document.links
  • document.images
  • document.forms
  • tr.cells
  • select.options

Dom 查找

Dom提供了如下的查找API

  • Document.getElementById()
  • Document.getElementByName()
  • Element.getElementByTagName()
  • Element.getElementByClassName()
  • Element.querySelector()
  • Element.querySelectorAll()

DOM 操作节点

Dom操作节点的API如下

  • Node.appendChild()
  • Node.insertBefore(newNode,refNode)
  • Node.replaceNode(newNode,oldNode)
  • Node.removeNode(node)
  • Node.clodeNode(node)
  • ChildeNode.before()、ChildeNode.after()、remove()、replaceWith()
  • normalize()/splitText()

属性操作

Js中属性分为固有属性(prop)和自定义属性(Attr)。

  • prop是固有的，有默认值。Attr需要自己指定
  • 固有属性(prop)指定后，会显示在标签上

固有属性可以通过Element.xx直接访问,可以直接设置

操作属性，可以使用Attibuites，本质上是NamedNodeMap

  • attributes[i/name] = attributes.item(i) = attributes.getNamedItem(name) -> Attr 有name(不能修改)和value属性
  • length - 长度
  • 修改 - attributes[i].value = ‘xxx’
  • 插入 - setNamedItem()
  • 删除 - removeNamedItem()

Element上提供了更加简单的Api

  • getAttribute - 获取
  • setAttribute - 设置 修改
  • removeAttribute - 删除

属性(Prop)有很多分类

  • 布尔属性- 值等于自己
  • 字符串属性
  • data属性
  • class属性
  • style属性
