Java

Java是什么？

Java是Sun公司推出的面向对象语言。

Java执行过程？

Java程序(.java)写好后，先编译成字节码(.class)，JVM转换字节码为二进制。

Java为什么能跨平台？

针对不同的OS提供了不同的JVM。

JVM、JRE、JDK区别？

  • JVM - Java虚拟机
  • JRE - 运行环境，包括JVM和Java类库(已经实现的Java程序)
  • JDK - JRE和开发工具(javac、java)

Java SE、Java EE、Java ME？

  • Java SE - 包括核心类库，比如数据库API、 输入输出、网络
  • Java EE - 包含Web开发所需内容，比如EJB、servlet、JSP、xml
  • Java ME - 移动端开发相关内容

这种划分方式，让每块内容更简介，更专业。

安装Java

  1. 下载、解压

    国内下载镜像，下载完成后，解压文件。

  2. 配置JAVA_HOME

    JAVA_HOME 表示JDK的目录，很多软件比如Eclipse、NetBeans、Tomcat都是通过JAVA_HOME来找到jdk位置。

  3. 配置PATH

    为了在任何地方使用javac、java等，要配置PATH。

  4. 配置CLASSPATH

    1.5以上不用配置，表示JVM去哪里读取字节码文件。
