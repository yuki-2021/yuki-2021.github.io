盒子模型

盒子模型

css引入了盒子模型，用来布局。每个元素都是一个盒子。

块元素和行内元素，盒子有如下区别：

  • 行内元素没有width和height
  • 行内元素的margin-top和margin-bottom失效
  • h1~h6、p、ul、ol、li、dl、dt、dd、table、form、div等是块元素
  • 修饰标签(b,strong,em,i,sup等)、img、a、表单元素(input、select、texearea)是行内元素

盒子相关属性如下

  • width|max-width|min-width: 长度 | 最大长度|最小长度
  • height | max-height | min-height: 高度 | 最大高度 | 最小高度
  • padding 内填充
  • margin 外填充
  • border 边框

width、height - 宽高

  • IE不支持max-height、min-heiht、max-width、min-width

border - 边框

  • 细分为border-width、border-color、border-style
  • 边框可以设置透明transparent

透明边框transparent

padding - 内填充

margin - 外填充

  • 绝对定位四个方向都是0，margin auto 实现居中
  • margin:auto 水平居中
  • 上下margin重合，取大的值

margin设置为auto居中

配合absolute设置绝对居中

上下margin会重叠取较大的，左右不会重叠。

不同的盒子模型

虽然有标准盒子模型，但是我大IE就是特例独行，也提出了IE盒子模型。

浏览器如何创建盒子

  • 声明了 <!doctype html> -> 标准盒子
  • 没声明 -> ie= IE盒子 其他浏览器 = 标准盒子

IE盒子 - 做IE就是要与众不同

  • 在IE盒子中 width = content + padding

display属性

display - 改变盒子类型(改变块or内联)

  • inline
  • block
  • inline-block
  • none - 不显示(不支持动画)
  • table
  • table-cell - 配合上面可以实现居中
