float浮动

Css中布局机制有很多

  • 普通流 - 默认机制，左到右，上到下
  • 浮动
  • 定位(position)
  • 多列布局
  • flex布局
  • grid布局(网格)

普通流

默认机制，元素 从左到右，从上到下。

浮动

Float ，元素脱离普通流，不脱离文字流。

  • 浮动后，脱离文档流，不脱离文字流 - 文字环绕

  • 多元素浮动，紧跟同类，过长下移

  • 脱离文档流，造成高度塌陷，影响后面元素

文字环绕 - 脱离文档流，不脱落文字流

紧跟浮动，过长下移

没浮动时，如下

ABCDE同时浮动，可以发现 -> 紧跟同类浮动的后面

当E过长，发现 -> 长度过长会下移

高度塌陷

开始如下

A浮动 -> A脱离文档流，其他三个按照文档流分布

ADC浮动 -> 容器高度完全塌陷

清除浮动

有如下方法

  • 父元素overflow:hidden
  • 父元素内部添加div,设置clear
  • 使用伪元素(::after)和clear
