背景和边框

نُشر في 2020-12-12 عُدل في 2020-12-13

背景和边框

border - 边框

  • 包括border-width、border-color、border-style
  • transparent设置透明边框

透明边框

background - 背景

包括如下内容

  • border-color - 背景色
  • border-image - 背景图片
  • border-position- 背景开始的位置
  • border-repeat - 背景重复
  • border-attachment - 偏移参照点

background-color - 背景颜色

  • 支持16进制(#fff)和rgba(x,x,x,x)
  • 默认起始于border

background-image - 背景图片

  • 默认起始于是border
  • 可以设置多个图片

background-position - 开始位置

  • 默认开始于左上角

。background-repeat - 重复方式

  • repeat - 默认 重复
  • repeat-x
  • repeat-y
  • no-repeat

background-attachment - 设置相对点

  • 影响的是background-position和background-repeated
  • scroll - 相对本元素 进行偏移和重复
  • fixed - 相对于body 进行偏移和重复

background - 背景

  • 顺序随意
  • 空格分隔

Css3 Moudle

border扩展了

  • border-radious - 圆角
  • border-image - 边框图片

background扩展了

  • background-orign
  • background-clip
  • background-size

新增加了box-shadow - 设置阴影

border-radious - 设置圆角

border-image - 设置背景图片

分为

  • border-image-slice - 图片
  • border-image-slice - 如何分隔
  • border-image-width - 占用多宽(大于border会往内缩)
  • border-image-outset - 往外扩展(不影响布局)
  • border-image-repeat - strech | repeat | round(重复，微略缩放) | space(重复，不够空白)

border-image 可以连写

background-orign - 图片起始位置

  • border-box - 默认 边框开始
  • padding-box- padding开始
  • content-box - 内容开始

background-clip - 图片显示范围

  • border-box - 显示在border上和内
  • padding-box - 显示在padding和content
  • content-box - 现实杂content中

background-size - 图片大小

  • auto - 默认大小
  • cover - 等比扩展，填充容器
  • contain - 扩展一边，填满

box-shadow - 阴影

语法 - box-shadow: x偏移 y偏移 模糊距离 扩展距离 [inset]

  • 模糊距离会单独多出来
  • 扩展距离会单独多出来

list-style

列表样式包括list-style-type、list-style-image、list-style-position。

  • list-style-type 表示列表标志的位置
  • list-style-image 把图片设置为列表标志
  • list-style-position: 别表标志位置

list-style-type取值如下。none是没有。

list-type-image 把图像设置为列表标志。图片显示原有大小。

list-style-posiiton设置列表标志的位置。

inside 内部
outside 外部

可以简写成list-style,不分顺序,空格分隔。list-style-image会覆盖list-style-type属性。
