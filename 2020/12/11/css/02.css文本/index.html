Posted on 2020-12-11 Edited on 2020-12-13

Css字体 - Css Text Module

字体可以设置如下属性

  • font-variant - 小写字母转换成小型大写字母

  • font-weight - 粗细

  • font-style - 是否倾斜

  • font-size - 字体大小

  • font-family - 类型

  • color表示字体颜色

font-variant - 是否是小型大写字母

  • normal 正常
  • small-caps 小写字母转换成小型大写字母

font-style -字体是否倾斜

  • normal - 正常
  • italic - 字体有斜体就倾斜，没有就不倾斜
  • oblique - 强制倾斜

font-weight - 设置粗细

  • normal=400 - 正常
  • blod=700 -粗体
  • lighter/bloder/数值

font-size - 字体大小

font-family - 字体 - 语法 font-family: f1,f2,...,s1...sn

  • f是字体,s是字体风格。当f都不成立，会应用s风格的字体。

常见字体风格有，详细参看

  • serif - 衬线体 - New Times Roman
  • sans-serif - 不衬线体 - Helvetica、Verdana
  • Monospace - 等宽(浓厚的盗版风格字体) - Lucida Console
  • Cursive - 手写
  • Fantasy - 不归入前面字体的字体

font - 上面内容可以缩写

说明

  • 空格分隔
  • 前三个顺序随意，后两个(fontSize、fontFamily)位置固定

颜色 - Css Color Module

color - 设置颜色，支持16进制和rgba(x,x,x,x)

文本 - Text moudle

主要包括如下属性

  • text-transform - 大小写转换

  • text-indent - 缩进

  • text-align - 对齐方式

  • line-height - 行高

  • vertical-align - 改变元素的baseline位置(inline、inline-block、img)

  • letter-spacing - 字母(汉字)之间间隔

  • word-spacing - 单词(空格分隔的文本)间隔

  • white-space - 回车和空格保留方式 以及 是否换行

text-transform - 大小写转换

  • none - 等于不设置
  • capitalize - 首字母大写
  • lowercase - 小写
  • uppercase - 大写

text-indent - 文字缩进

text-align - 文字对齐

  • left - 左对齐
  • right - 右对齐
  • center - 居中
  • start - 设置了dir属性或者direction，位于开始方向
  • end - 设置了dir或者direction，位于结束方向
  • justify - 两端对齐，不处理最后一行、br强制换的行

当文本只有一行，这行是最后一行，不处理。

加个br，变成两行，但是text-align不处理强制打断的行

可以如下解决，详细参看

line-height - 设置行高

  • 只设置line-height,不设置height，可以把height撑起来

vertical-align - 设置basline对齐方式

元素baseline位置

  • 内联元素 - 位于文字a的下端(英文四线格，a下面的那个)
  • img - 位于边缘
  • inline-block,里面有内联元素 - 最后一行的基线
  • inlineBlock，没有内容 - margin底部边缘

vertical可以设置的值

  • sub - 基线下移
  • super - 基线上移
  • middle - 基线中间
  • top - 基线移动到最高元素的顶部
  • bottom - 基线移动到最低元素底边
  • text-top - 元素和内容顶部对齐
  • text-bottom - 和底部内容区域对齐

word-spacing - 单词之间距离

letter-spacing - 字母之间的距离

white-space - 空格合并 回车保留 是否换行

  • normal - 空格合并 回车删除 换行
  • nowrap - 空格合并 回车删除 不换行
  • pre - 空格不合并 回车保留 不换行
  • pre-wrap - 空格不合并 回车保留 换行
  • pre-line - 空格合并 回车保留 不换行

文本装饰- Text decoration

如下

  • text-decoration-line - 装饰位置
  • text-decoration-color - 颜色
  • text-decoration-style - 线条样式

text-decoration-line - 装饰线位置

none 没有
underline 底部
overline 头部
line-through 中间

text-decoration-color - 线条颜色

  • 不给 - 默认黑色

text-decoration-style -线条样式

solid 实线
double 双线
dotted 点
dashed 点
wavy 波浪线

word-wrap - 单词换行

  • normal: 单词放不下，直接溢出
  • break-word: 单词放不下，放到下一行，并且换行。

word-break - 更全面的换行

最开始，大家都用word-wrap,我们喜闻乐见的ie觉得word-wrap不行，功能太少了，于是做了word-wrap做了plus版本，叫做word-break。

  • normal - 同上
  • break-word - 同上
  • keep-all - 直接全部一行
  • break-all - 单词直接追加，碰到边界直接截断
