css选择器

CSS简介

CSS，层叠样式表，用来给html页面添加样式。

1996年，W3C发布了CSS1。

1998年，W3C发布了Css2。因为一次发布太多内容，到2011年各个厂商才基本实现支持。

CSS33把内容分为小模块，每年会更新一些模块(snapshots版本)，比如2007、2010、2015、2017。目前模块还没全部完成(GA版本)。等模块全部完善，Css3就会正式发布了。

各个模块情况css模块状态

语法

语法包括两部分

  • 选择器
  • 样式

Css引入

引入如下

  • 四种方式 - 内联、内部样式表、外部样式表、导入⓷
  • 优先级 - 内联 > 内部 = 外部 = 导入⓷
  • 后三个距离越近，优先级越高

Css选择器

基本选择器

通配符选择器 *， 权重 0，用来做初始化(cssReset)。

标签选择器，权重1。

类选择器.，也叫Class选择器，权重10。

Id选择器#，权重100。

关系选择器

通用后代选择器，选中所有的后代。

直接后代选择器，选择直接后代。

兄弟选择器，选择紧邻的兄弟，**(紧邻&&某类型)同时满足。**

⓷所有兄弟选择器-css3

分组选择器

多个分组同时实现样式。

伪类选择器

首先是

  • :link-访问前、:visted-访问后、:hover-悬浮、:active-激活(按住鼠标不松手)、
  • 书写顺序 - link > visted > hover > active
  • <a>四个都有，其他只有active、active

:focus - input被聚焦

Css优先级

样式生效规则

  • !important最先生效
  • 一般来说第二高的是行内1000
  • 优先级高的生效
  • 优先级相同，按照导入计算，距离近优先

优先级别最高的是 !inportant

权重计算规则是

  • id=100 class=10 标签=1

  • 直接累加
