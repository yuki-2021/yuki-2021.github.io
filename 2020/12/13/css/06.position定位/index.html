Position定位

Position定位分为

  • static - 静态定位

  • relative -相对定位

  • absolute - 绝对定位

  • fixed - 固定定位

  • sticky - 磁铁定位

static静态定位

让元素回归自然流

  • top、left、right、bottom 失效

开始，ADBC相对定位。

B静态定位 -> 位置属性失效

relative - 相对定位

  • 用于设置absolute的参照点
  • 以前位置保留
  • top、left、right、bottom 相对以前位置移动
  • 可以移动浮动元素

开始如下

B设置相对定位 -> B相对自移动，位置得到保留

static - 绝对定位

  • 脱离文档流,位置直接不保留
  • 有relative就参照relative,没有参照body
  • 宽度设置**%**相对父元素
  • 居中(lrbt=0,margin=auto或者l50t50+traslate)

开始如下

B绝对定位 ->脱离文档流，位置不保留，相对body

设置居中

固定定位 - fixed

fixed是固定定位，相对于视口固定

  • 相对视口固定
  • 脱离文档流
  • 宽度相对body

磁贴定位 - sticky

sticky是`磁贴定位。

  • 元素能显示在视口中，显示在正常的位置
  • 元素移动，出了视口，出现在sticky固定的位置
