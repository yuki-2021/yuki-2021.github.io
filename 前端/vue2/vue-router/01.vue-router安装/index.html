vue-router安装

نُشر في 2021-09-14

直接引入

直接引入https://unpkg.com/vue-router/dist/vue-router.js。

vue-cli安装

可以在vue create project-name的时候，选择vue-router。

另外，对于Vue3，可以用vue add router来安装vue-router。

  • 会安装vue-router
  • 会在生成src/router/router.js

vue2比较麻烦

  • 安装npm install vue-router --save
  • 创建src/router/index.js文件
  • 在/src/main.js中导入
