vuex介绍

vuex类似于全局对象，用于在不同组件中传值。

包括如下概念

  • state - 类似于data，表示全局存储的数据。
  • getters - 类似于computed。当state变化，getters会从新计算
  • mutations - 用于更新state
  • actions - 异步更新state。
  • modules - 可以把state、getters、mutations、actions分割成不同模块。

vuex安装

如果，你用的是vue-cli，在vue create xxx的时候，选择vuex就OK。

如果你用了vue3，用vue add vuex，可以安装vuex。

  • 会安装vuex
  • 会生成@/store/index.js文件

如果你是vuex2

  • 用npm install vuex --save安装
  • 在@/store/index.js创建Vuex.Store创建store
  • 传给vue实例
