    File

    File类表示文件或目录。

    静态属性

    pathSeparator - 目录分隔符号

    separator - 路径分隔符号

    静态方法

    createTemplateFile(prefix,suffix,[path]) - 创建临时文件

    listRoots() - 获取所有根目录

    构造方法

    • 用绝对路径，File指向绝对路径位置
    • 使用相对路径，File指向user.dir + path的位置
    • 创建了File不表示创建了文件，仅仅表示在内存中创建了File类型的对象，这个对象有对这个路径操作的能力

    实例方法

    获取属性

    获取属性 - canRead() / canWrite() / canExecute() / lastModified() / getName()

    设置属性

    设置属性 - setExecuteable() / setReadable() / setWriteable() / setReadOnly() / setLastModified()

    比较和判断

    比较和判断 - exists() / compareTo() / isAbsolute() / isFile() / isDirectory() / isHidden()

    获取路径

    获取路径 - getPath() / getAbsolutePath() / getCanconicalPath() / getParent()

    转换

    转换 - toURI() / toPath() / toString()

    遍历

    遍历 - list([filenameFilter]) / listFiles([filenameFilter]) / listFiles(fileFilter)
