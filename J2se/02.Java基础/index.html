    变量和常量

    任何编程语言都有变量和常量概念。

    命名规则

    • 字母(a-z)、数字(0-9)、下划线(_)、美元符号($)

    • 不能以数字(0-9)开头

    常量 - final
    • 常量必须赋值一次，不用立即赋值
    • 最好声明时候直接赋值
    • 有局部常量、属性常量、静态常量

    数据类型

    数据类型分为基本数据类型和对象类型。

    对象类型是引用类型，由基本数据类型组成。

    基本数据类型分类
    • 整数类型 - byte(1) short(2) int(4) long(8)
    • 小数类型 - float(1.4E-453.4E10+38) double(4.9E-3241.7E+308)
    • 字符类型 -
    • 布尔类型 - true/false
